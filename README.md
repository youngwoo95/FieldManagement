# PlantManagement

공장 설비/층/조명 정보를 관리하는 WPF 기반 관리 시스템입니다.

## 주요 기능
- 설비 상태 조회 및 모니터링
- 층(Floor) 정보 관리
- 설비 상태/색상 관리
- 고객사 관리
- 수주 관리
- 작업 관리

## 기술 스택
- .NET 10
- WPF
- Entity Framework Core
- Dapper

## 실행 방법
```bash
dotnet restore
dotnet build
dotnet run --project PlantManagement
